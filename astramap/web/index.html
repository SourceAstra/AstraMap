🚀 AstraMap 代码地图

核心优势

  • 95%+ 语义精度 — SCIP 编译器级索引，区分重载/泛型
  • 60-95% Token 节约 — 单次调用替代多次 grep+Read
  • 毫秒级更新 — 文件变更 50ms 内同步
  • 条件编译支持 — C++/Go/Rust 条件分支动态投影

工作原理

源码 → SCIP 高精度索引 + Tree-sitter 实时补丁 → SQLite 知识图谱 → MCP/HTTP API → 本地可视化与工具客户端

🚀 2 分钟部署指南 (Linux)

📁 第一步：加入 PATH
mkdir -p ~/bin cp ./amap ~/bin
echo 'export PATH="$HOME/bin:$PATH"' >> ~/.bashrc source ~/.bashrc
📂 第二步，进入所在项目
# 一键注册 MCP + 规则文件到 Claude Code等
amap install
# 默认模式：自动检测变成工具，并生成代码地图所使用 SCIP
amap index
🤖 第三步，打开 Claude Code
测试代码地图使用可以使用/amap 斜杠命令
或 MCP 服务器自动集成
🖥️ 第四步，进入所在项目，可使用代码看函数调用链图
amap dashboard
默认端口 3000，冲突自动+1
访问: http://localhost:3000
